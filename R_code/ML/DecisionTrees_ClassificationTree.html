  2. the number of observations in that branch,
  3. the deviance,
  4. the overall prediction for the branch (Yes or No).

Split the observations into a training set and a test set (200/200)

set.seed(2)
train <- sample (1: nrow(Carseats), 200)
Carseats.test <- Carseats[-train ,]
High.test <- High[-train ]
tree.carseats <- tree(High ~ . -Sales ,data = Carseats , subset = train )
tree.pred <- predict(tree.carseats ,Carseats.test ,type ="class")
table(Predict = tree.pred, True = High.test)
##        True
## Predict  No Yes
##     No  104  33
##     Yes  13  50
mean(tree.pred == High.test)
## [1] 0.77

  • predict() function can be used for this purpose. In the case of a classification tree, the argument type=“class” instructs R to return the actual class prediction.

  • correct predictions: 77%

  • pruning the tree by cv.tree() cross-validation, the optimal level of tree complexity.

  • FUN = prune.misclass: indicate that we want the classification error rate; default is deviance.

  • size: the number of terminal nodes of each tree

  • dev: error rate

  • k: cost-complexity parameter

The tree with 9 terminal nodes results in the lowest cross-validation error rate, with 50 cross-validation errors

par(mfrow =c(1,2))
plot(cv.carseats$size ,cv.carseats$dev ,type="b")
plot(cv.carseats$k ,cv.carseats$dev ,type="b")

  • We now apply the prune.misclass() function in order to prune the tree to prune. Obtain the 9-node tree.
prune.carseats <- prune.misclass(tree.carseats ,best = 9)
plot(prune.carseats )
text(prune.carseats ,pretty =0)

tree.pred <- predict(prune.carseats ,Carseats.test ,type= "class")
table(Predict = tree.pred, True = High.test)
